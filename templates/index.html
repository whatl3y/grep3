The Ecosystem

Purpose-built microservices

Each app is independent, containerized, and designed for a specific use case. Mix and match to build your own infrastructure.

exec-engine API

Execute any code in Docker containers. Push git repos via Web3-authenticated endpoints and run them securely with x402 payment gates.

Docker Git Server x402 S3 Logs

tornado-api Privacy

Privacy-preserving cryptocurrency transactions via Tornado Cash. Generate notes, deposit funds, and withdraw anonymously across multiple networks.

ZK Proofs 8+ Networks Anonymous

merkletree-api Crypto

Generate and manage Merkle trees from raw data or files. Background job processing with proof generation for airdrops and allowlists.

OpenZeppelin CSV/Files Proofs

liquidity DeFi

Uniswap V3/V4 concentrated liquidity visualization. Real-time heatmaps, price curves, and depth charts for any pool.

V3 + V4 Heatmaps React

remix AI

Transform any website into stylized versions using AI. Capture full pages and restyle them with OpenAI—8-bit retro and more.

OpenAI Playwright 8-bit

ncaab-predict ML

NCAA Basketball prediction using machine learning. Point spreads, win probabilities, and risk analysis powered by XGBoost.

XGBoost Python 15K Games

grep3 CLI Tool

The unified command-line interface for all grep3 APIs. Install globally via npm and interact with merkletree, tornado, execution engine, and more—all from your terminal.

npm install Namespaced Commands Interactive Prompts Web3
# Install globally
npm install -g @grep3/cli
# Generate a merkle tree
grep3 merkletree generate-file --file allowlist.csv
Philosophy

No accounts. Just crypto.

grep3 services authenticate via Web3 wallets and monetize through x402 payment rails. No emails. No passwords. No vendor lock-in.

1

Connect Wallet

Authenticate with your Ethereum wallet. Your address is your identity—no accounts to create or manage.

2

Pay Per Use

Services are gated with x402 micropayments. Pay only for what you use, directly on-chain.

3

Own Your Data

No centralized databases storing your information. Outputs go to your storage, keys stay with you.
